# 🖼️ Image Editor

A simple and interactive browser-based Image Editor built using **HTML**, **CSS**, and **JavaScript**. It allows users to upload an image, apply filters, reset changes, and download the edited image.

## ✨ Features

- 📁 Upload images from your device
- 🎨 Apply multiple image filters
- 🎛️ Preset filter options
- 🔄 Reset all edits with one click
- 💾 Download the edited image
- 📱 Responsive and clean user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- HTML5 Canvas API
- Remix Icons

### Open the project

Simply open `index.html` in your browser.

Or use the VS Code Live Server extension for a better development experience.

## 🎮 How to Use

1. Click **Choose Image**.
2. Select an image from your computer.
3. Apply filters or presets.
4. Click **Reset** to restore the original image.
5. Click **Download** to save the edited image.
